## 💻 About Me

I'm a backend developer passionate about building **scalable, well-structured, and maintainable systems**. I enjoy working on the server side of applications, focusing on **performance, architecture, and clean code**.

My main stack revolves around **Java and the Spring ecosystem**, where I build **RESTful APIs**, design backend architectures, and implement business logic for real-world applications. I have experience working with tools commonly used in modern backend development, including **API documentation, database integration, and version control**.

Beyond Java, I also have experience with **Python and Django**, which I use for building web backends and experimenting with different development approaches. I work with **SQL databases** for data modeling, queries, and persistence, and I’m comfortable integrating backend services with databases in a clean and efficient way.

I also have familiarity with **JavaScript and TypeScript/TSX**, which helps me understand full-stack workflows and collaborate better when backend systems interact with frontend applications.

What motivates me most in programming is **solving real problems, designing efficient systems, and continuously improving my technical skills**.

I enjoy creating **personal projects to explore new technologies, improve my architecture skills, and practice building production-like systems.**
