Backend Developer

💻 Sobre mim

Sou desenvolvedor backend apaixonado por construir sistemas bem estruturados, escaláveis e de fácil manutenção. Tenho grande interesse na parte de lógica, arquitetura e funcionamento interno das aplicações, buscando sempre escrever código limpo e organizado.

Minha principal stack é Java com o ecossistema Spring, onde desenvolvo APIs REST, implemento regras de negócio e trabalho na estruturação de aplicações backend voltadas para problemas reais. Gosto de trabalhar com boas práticas de desenvolvimento, organização de código e construção de sistemas que sejam fáceis de evoluir e manter.

Também possuo experiência com Python e Django, que utilizo para desenvolver aplicações web e explorar diferentes abordagens no desenvolvimento backend. Além disso, trabalho com bancos de dados SQL, realizando modelagem de dados, criação de consultas e integração eficiente entre aplicação e banco.

Tenho também familiaridade com JavaScript e TypeScript (TSX), o que me permite entender melhor a integração entre backend e frontend e colaborar de forma mais eficiente em projetos completos.

Sou uma pessoa curiosa e sempre busco aprender novas tecnologias e aprimorar minhas habilidades técnicas. Gosto de desenvolver projetos pessoais, principalmente voltados para backend, que simulam aplicações reais e me ajudam a evoluir em arquitetura de software, APIs e design de sistemas.

Atualmente meus principais focos são:

Desenvolvimento backend com Java e Spring

Construção de APIs REST

Modelagem e manipulação de bancos de dados SQL

Estudo de arquitetura de software e boas práticas

Desenvolvimento de projetos que simulam sistemas reais

Estou sempre buscando evoluir como desenvolvedor, melhorar a qualidade do meu código e construir projetos cada vez mais completos.
